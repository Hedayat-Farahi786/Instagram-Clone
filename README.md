# Instagram Clone
![https://github.com/Hedayat-Farahi786/Instagram-Clone](http://shimmeringsoul.weebly.com/uploads/1/3/0/3/13031044/1-xkmi4fb5vws6-my7b22lza_1.png)

## Tech Stack

* Frontend
  * Next JS / React JS
  * Tailwind CSS
  * Recoil
* Backend
  * Firebase / Node JS
* Login
  * NextAuth 
  * Google OAuth

## Configuration / Setup

First, install the required dependencies

```bash
npm install
# or
yarn install
```

Serve the site locally

```bash
npm start
# or
yarn start
```
